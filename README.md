# Path-Planning-Algorithms---Algoritmos-de-Planeamento-de-Rotas
Here are some path planning algorithms that can be applied according to the intended context. // Aqui estão apresentados alguns algoritmos de planeamento de caminhos, podem ser aplicados comforme o contexto pretendido. 
The algorithms are A* in Python and C#, Theta* in Python, and FloodFill in C#. The heuristics used were Euclidean and Manhattan. The algorithm codes include a section dedicated to graphical visualization. // Os algoritmos são A* em Python e em C#, Theta* em Python e FloodFill em C#. As heurísticas usadas foram a euclidiana e a de Manhattan. Há nos códigos dos algoritmos uma parte que é dedicada a visualização gráfica do algoritmo.
